Features which can be added

Those are for example city coordinates lookup. It lets you enter a city name and returns the corresponding coordinates.

Also an interesting idea is cascading multiple images to get a bigger view. But this requires high precision coordinate computation.

One could check the cloudiness value and search for the closest non-cloudy image in time. Must be optional.

And of course it would be nice if your current position could be looked up.

But first, correct date computation must be implemented. February has only ~28 days.
